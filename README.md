# Primeiros passos com o React

Olá pessoa! Espero que você que tá lendo isso esteja de boa 😎

Bom, esse projeto teve o intuito de aprender a mexer no React, Api e tudo que envolve os dois lados: front e back end.
Gostei bastante de fazer esse projeto graças as duas videoaulas que assisti e consegui pegar um pouco da ideia. Veremos isso nos próximos projetos que irei fazer... 

É um projeto simples, mas foi bastante esclarecedor. 

"Ok, eu sei que foi seu primeiro projeto usando React, Api e essas coisas, mas... o que foi que tu desenvolveu?"
Então, o que foi proposto nas videoaulas e eu fiz foi um site de Cadastrar usuários. Bem básico mesmo. No diretório **src** estão os arquivos do React, Html e Css e afins (coisas do front-end) e já o diretório **api** estão os arquivos do servidor (back-end e banco de dados).
